# Problem solving

**Author:** Fazal  
**Started:** April 20, 2026  

## What this is

This repo is a **personal check-in** on how consistently I show up to solve problems, not a showcase. It is partly a small assessment of my own follow-through: whether I make time to work through something hard instead of only planning to.

From this day on, I will try to solve **at least one problem a day**.

## Why it exists

When AI took off, I found myself **coding a lot less by hand**—a lot of what I shipped was effectively AI-generated. In my day-to-day that was easy, but when I did get **stuck on a real problem and tried to untangle it myself**, I was struck by how **hard** even “simple” things had become. That wake-up is what nudged me back into **solving different problems and saving them here**: so I have a **record that I was here**, that I sat with the problem and got something down.

## About the code

**These are not the best solutions you will find**—they are the **best I could do in the moment**, with the time and headspace I had. I may come back to the same problems later to tighten logic, try another approach, or compare with how others think about them. This repo is allowed to be a **progress trail**, not a final exam.
